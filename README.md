# linktree-clone
**Página de links estilo Linktree desenvolvida com HTML e CSS para prática de layouts responsivos.**


## Tecnologias Utilizadas:
* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização, fontes personalizadas e efeitos de transição nos botões.
* **SVG/Icons:** Implementação de ícones de redes sociais.
